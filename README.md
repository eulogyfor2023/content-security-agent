# content-security-agent
Content Security Agent 是一款 AI 驱动的数据分析工作台，面向内容安全运营与数据分析岗位设计。它集成了 AI 内容安全分析（自动识别风险等级、评分、标签并生成差异化运营策略）、统计异常检测（Z-score 突增/突降识别、高风险内容聚集预警）、用户行为分析（漏斗转化、留存率、行为画像）、自动报表生成（日报/周报/月报 + AI 运营建议，支持一键推送到微信）、智能风控规则引擎（8 条可配置规则，支持自动告警/标记/限流）以及日程提醒系统（提前 6 小时微信推送），全部功能通过深色主题 ECharts 可视化仪表盘统一呈现，7 个标签页覆盖数据分析岗从风险监控、用户洞察、策略制定到报表产出的完整工作流。
